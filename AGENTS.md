# AGENTS.md

## Project Overview

This is a Next.js App Router single-page landing website for a Chinese car wash and detailing shop named 澄光汽车洗护中心.

The page promotes exterior washing, interior cleaning, paint care, odor removal, package pricing, and appointment booking.

## Structure

- `src/app/page.tsx`: Main landing page route.
- `src/app/layout.tsx`: Root layout and metadata.
- `src/app/globals.css`: Global visual styles.
- `src/components/BookingForm.tsx`: Client-side demo booking form.
- `public/assets/logo-car-wash.svg`: Lightweight logo asset.

## Design Notes

Keep the brand direction fresh, clean, premium, and friendly. The visual palette uses navy, blue, aqua teal, warm orange, and cream. Avoid changing it into a generic purple SaaS design unless requested.

## Commands

```text
npm install
npm run dev
npm run build
npm run lint
```

The booking form is demo-only and does not submit to a backend.
